# 圣地巡礼角色实景合成器 (Anime In Real Life)

在圣地巡礼时，将动漫角色与实景照片完美融合的 Web 应用。

## ✨ 主要功能

### 1. 本地提取 (Local Extraction)
*   **智能抠图**：使用浏览器端 AI 模型 (IS-Net) 自动从动漫截图中提取角色。
*   **实时合成**：将提取的角色叠加到实景照片上。
*   **自由调整**：支持调整角色的大小、位置、透明度及边框样式。
*   **隐私安全**：所有处理均在本地完成，无需上传图片到服务器。

### 2. AI 生成 (Nano Banana Pro) 🍌
> [!IMPORTANT]
> 此功能需要使用您自己的 Google AI Studio API Key，且生成图片会产生费用（约 $0.24美元/张）。

*   **智能融合**：利用 Nano Banana Pro 的强大多模态能力，将动漫角色与实景背景自然融合。
*   **光影适配**：AI 自动调整角色的光影和色调，使其与环境更协调。
*   **自定义提示词**：支持自定义 Prompt 微调生成效果。
*   **双重下载**：提供“对比图”和“纯实景生成图”两种下载选项。

## 🚀 快速开始

### 开发环境
1.  安装依赖：
    ```bash
    npm install
    ```
2.  启动开发服务器：
    ```bash
    npm run dev
    ```

### 使用说明
1.  **上传图片**：点击上半部分上传动漫截图，下半部分上传实景照片。
2.  **选择模式**：
    *   **本地提取**：点击“提取角色”，等待模型加载并自动抠图。
    *   **AI 生成**：切换到 AI 标签页，输入 Gemini API Key，点击“开始 AI 生成”。
3.  **保存结果**：点击下载按钮保存合成后的图片。

## 🛠️ 技术栈
*   **前端**：Vanilla JS, Vite, Less
*   **AI 模型**：
    *   本地：IS-Net (via Transformers.js)
    *   云端：gemini-3-pro-image-preview 🍌
*   **部署**：GitHub Pages (自动化构建)
